2. SELECT (Studi Kasus Hasil)

Read

Cheat Sheet Syntax

Operator Pembanding

  • = : Sama dengan
  • != : Tidak sama dengan
  • > : Lebih besar
  • < : Lebih kecil
  • >= : Lebih besar sama
  • <= : Lebih kecil sama

Operator Logika

  • AND : Syarat harus benar semua
  • OR : Salah satu syarat benar
  • NOT : Kebalikan (Negasi)
  • IN : Mencocokkan list
  • LIKE : Pencarian teks
  • BETWEEN : Rentang nilai

Fungsi Agregasi

  • COUNT() : Hitung baris data
  • SUM() : Total penjumlahan
  • AVG() : Nilai rata-rata
  • MAX() : Nilai tertinggi
  • MIN() : Nilai terendah

A. Contoh Kasus: Filter Nilai (WHERE > 80)

Tampilkan siswa yang nilainya di atas 80.

SELECT name, score FROM students WHERE score > 80;

C. Hitung-hitungan (Aggregation)

Total Siswa:

SELECT COUNT(*) FROM students;
Result: 10

Rata-rata Nilai:

SELECT AVG(score) FROM students;

Hitung jumlah siswa di setiap kelas.

5. JOIN (Menggabungkan Data)

Inner vs Left

Bagaimana cara menampilkan Nama Siswa beserta Nama Kelasnya? Ada 2 cara paling umum yang hasilnya berbeda jika datanya tidak lengkap.

A. INNER JOIN (Standar)

SELECT students.name, classrooms.name AS class_name
FROM students
JOIN classrooms ON students.classroom_id = classrooms.id;

Hanya menampilkan data yang punya pasangan lengkap. Siswa tanpa kelas TIDAK akan muncul.

B. LEFT JOIN (Tampilkan Semua Kiri)

SELECT students.name, classrooms.name AS class_name
FROM students
LEFT JOIN classrooms ON students.classroom_id = classrooms.id;

6. MULTI JOIN (Gabungan 3 Tabel)

Complex Read

Tantangan: Bagaimana jika kita ingin melihat Nama Siswa, Nama Kelasnya, DAN Nama Wali Kelasnya?
Asumsi: Tabel classrooms punya kolom teacher_id yang terhubung ke tabel baru bernama teachers.

Students ➜ Classrooms ➜ Teachers

SQL Query:

SELECT
  students.name AS siswa,
  classrooms.name AS kelas,
  teachers.name AS wali_kelas
FROM students
JOIN classrooms ON students.classroom_id = classrooms.id
JOIN teachers ON classrooms.teacher_id = teachers.id;
