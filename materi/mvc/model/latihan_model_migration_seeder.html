Tantangan 3 Tabel

Tantangan Relasi Database

Misi Anda adalah membangun database yang saling terhubung tanpa error.

1 Pahami Struktur

Categories

(Induk)

  • id (PK)
  • name
  • slug

Products

(Anak dari Categories)

  • id (PK)
  • category_id (FK)
  • name, price, stock

Reviews

(Anak dari Products)

  • id (PK)
  • product_id (FK)
  • rating (float)
  • comment

2 Daftar Tugas (To-Do List)

Tugas 1: Fix Migration Order (Timestamp)

Masalah: Migration products dijalankan duluan daripada categories.

Contoh Nama File Migration:

2024_03_01_100000_create_products_table.php (Jam 10)

2024_03_01_100500_create_categories_table.php (Jam 10 Lewat 5)

Solusi: Rename file category. Ubah angkanya menjadi lebih kecil (lebih tua) dari product.
Misal ubah menjadi: 2024_03_01_090000_... (Jam 9 pagi).

Tugas 2: Buat Fitur Review

Buat Model & Migration Review.
Tambahkan kolom $table->foreignId('product_id')->constrained()->cascadeOnDelete();
Tips: Lihat kembali Modul Materi untuk aturan penamaan foreignId.

Tugas 3: Seeding Relasi

Buat ReviewSeeder yang mengisi 3 ulasan untuk produk ID 1.
Jalankan php artisan migrate:fresh --seed dan pastikan tidak ada error merah di terminal.
