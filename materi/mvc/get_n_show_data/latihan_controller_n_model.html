Misi Sesi 11: The Data Hunter

Tugas Anda adalah mengambil data spesifik dari Database dan membuktikannya dengan dd().

0 Persiapan

Pastikan Anda sudah punya Controller:

php artisan make:controller ProductController

Misi 1: Cek Stok Gudang (Basic)

Ambil SEMUA data produk, tapi urutkan dari yang paling baru diinput.

Clue: Gunakan latest() dan jangan lupa get().
Goal: Saat di-refresh, data dengan ID terbesar harus ada di paling atas (index 0).

Misi 2: Barang Langka (Where)

Boss minta data produk yang stoknya kurang dari atau sama dengan 10.

Clue: Gunakan where('stock', '<=', 10).
Goal: Pastikan di layar hitam (dd), kolom stock pada semua item tidak ada yang lebih dari 10.

Misi 3: Pencarian Spesifik (Find)

Cari produk dengan ID 1.

Clue: Gunakan find(1).
Goal: Hasilnya harus berupa Object (bukan Array). Tidak ada kurung siku [] pembungkus di tampilan dd().

Misi 4: Sambungkan Kabel (Relasi)

Ambil semua produk, tapi sertakan data Category-nya (Eager Loading).

Langkah A (Model): Pastikan di Product.php sudah ada fungsi category() dengan belongsTo.
Langkah B (Controller): Gunakan with('category') sebelum get.
Goal: Di tampilan dd(), klik panah kecil pada salah satu item produk -> cari properti relations -> pastikan ada isinya category.
*Ingat: Gunakan dd() di setiap langkah untuk memverifikasi hasil sebelum lanjut ke misi berikutnya.
