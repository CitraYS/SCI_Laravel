! Setup Tailwind (CDN)

Cara menggunakan Tailwind tanpa install (Cocok untuk Belajar/Prototyping):

Cukup copy script di bawah ini dan paste ke dalam tag <head> di file HTML atau Blade Layout Anda.

<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">

  <!-- TEMPEL SCRIPT INI -->
  <script src="https://cdn.tailwindcss.com"></script>
</head>
<body>
  ...
</body>
</html>
