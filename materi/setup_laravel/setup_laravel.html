Memulai Project Laravel

Panduan instalasi dan persiapan lingkungan kerja (Environment Setup) dari nol sampai server jalan.

01 Install via Composer

Buka VS Code, lalu buka Integrated Terminal. Kita akan mendownload Laravel langsung dari sini.

Tips Shortcut:

Tekan Ctrl + ` (Backtick) atau Ctrl + J untuk membuka/tutup terminal dengan cepat.

Terminal (VS Code)
bash / powershell
user@vscode:~/Desktop$ composer create-project laravel/laravel toko-online

Hasil di Komputer:

Desktop/
└── toko-online/ <-- Folder baru akan muncul

02 Masuk ke Folder

Setelah download selesai, terminal kita masih berada di "luar" folder project. Kita harus masuk ke dalamnya agar bisa memerintah Laravel.

Penting:

Jika kamu lupa langkah ini, perintah artisan di langkah selanjutnya akan error: "Could not open input file: artisan".

Terminal (VS Code)
user@vscode:~/Desktop$ cd toko-online
user@vscode:~/Desktop/toko-online$ _

03 Nyalakan Server

Laravel punya server lokal bawaan. Kita tidak perlu setting XAMPP ribet-ribet secara manual. Cukup panggil "Artisan".

Tips:

Jangan tutup terminal ini selama kamu sedang ngoding. Kalau ditutup, website-nya mati.

Terminal (VS Code)
server running
user@vscode:~/.../toko-online$ php artisan serve

INFO Server running on [http://127.0.0.1:8000].
Press Ctrl+C to stop the server
http://localhost:8000
Laravel v10.x.x (PHP v8.x)

04 Cek Folder Project

Pastikan kamu membuka folder projectnya (bukan folder luarnya) di VS Code supaya fitur Autosuggestion berjalan lancar.

.env

File rahasia. Tempat setting Database & Password.

vendor/

Isi "otak" Laravel yang didownload Composer. Jangan diedit!
