Sesi 12: View & Asset Management

Frontend Basics

1. Konsep: Etalase & Dekorasi

Data dari Controller perlu ditampilkan di View (HTML).

Agar tidak polos, kita butuh CSS. Di Laravel, file gambar, CSS, dan JS publik harus disimpan di folder khusus bernama public.

2. Membuat File View

Lokasi: resources/views/products/index.blade.php

Kenapa .blade.php?
Agar kita bisa menggunakan fitur pintar Laravel seperti @foreach dan {{ $variable }}. Kalau cuma .php biasa, codingnya akan ribet.

3. Update Controller (Melempar Data)

Hapus dd(), ganti dengan return view.

public function index()
{
    $shopName = "Toko Sembako Jaya"; // Data Tunggal (String)
    $products = \App\Models\Product::all(); // Data Banyak (Array)
    
    return view('products.index', compact('shopName', 'products'));
}

4. Manajemen Asset (CSS Manual)

Di Laravel, kita tidak boleh menyimpan CSS di dalam folder resources atau app jika ingin diakses langsung oleh browser.

Langkah A: Buat File CSS

Buat file di public/css/style.css:

my-project/
├── public/
│ ├── css/
│ │ └── style.css
│ └── ...

Isi File style.css:

body {
    font-family: sans-serif;
    background-color: #f4f4f4;
    padding: 20px;
}

.container {
    max-width: 800px;
    margin: 0 auto;
}

.card {
    background: white;
    padding: 15px;
    margin-bottom: 10px;
    border-radius: 8px;
    border: 1px solid #ddd;
}

.title {
    color: #2563eb; /* Biru */
    font-weight: bold;
    font-size: 1.2rem;
}

Langkah B: Panggil dengan helper asset()

Di dalam file index.blade.php bagian <head>:

<link rel="stylesheet" href="{{ asset('css/style.css') }}">

Fungsi asset(): Otomatis membuat URL lengkap (http://localhost:8000/css/style.css). Jangan tulis manual path-nya!

Refresher: Dasar Penamaan CSS

1. Tag Selector Global
body { ... }

Mempengaruhi SEMUA elemen <body> tanpa terkecuali.

2. Class Selector (.) Paling Sering
.card { ... }

Mempengaruhi elemen dengan class="card". Bisa dipakai berulang kali.

3. ID Selector (#) Unik
#header { ... }

Hanya untuk SATU elemen unik dengan id="header".

5. Blade Syntax 101 (Dasar)

Sebelum masuk ke looping data, pahami dulu cara menampilkan data satuan.

A. Menampilkan Variabel (Echo)

PHP Biasa: <?php echo $shopName; ?>
Laravel Blade: {{ $shopName }}

*Kurung kurawal ganda {{ }} otomatis mengamankan data dari serangan XSS.

6. Coding View Lengkap (HTML + Blade)

<!DOCTYPE html>
<html>
<head>
    <title>Toko Online</title>
    <!-- Panggil CSS External -->
    <link rel="stylesheet" href="{{ asset('css/style.css') }}">
</head>
<body>

    <!-- CONTOH 1: Menampilkan Data Tunggal -->
    <h1>Selamat Datang di {{ $shopName }}</h1>

    <hr>

    <!-- CONTOH 2: Menampilkan Banyak Data (Looping) -->
    <h2>Daftar Produk</h2>

    <div class="container">
        @foreach($products as $product)
            
            <div class="card">
                <h3 class="title">{{ $product->name }}</h3>
                <p>Rp {{ number_format($product->price) }}</p>
                
                @if($product->stock < 5)
                    <small style="color: red;">Stok menipis: {{ $product->stock }}</small>
                @else
                    <small>Stok: {{ $product->stock }}</small>
                @endif
            </div>

        @endforeach
    </div>

</body>
</html>
