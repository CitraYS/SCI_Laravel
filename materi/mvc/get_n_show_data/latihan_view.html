Misi Sesi 12: The Frontend Architect

Tugas: Mengubah data mentah menjadi katalog produk yang rapi dengan CSS sendiri.

Target Visual (Expected Output)

Gol akhirnya adalah halaman yang memiliki kartu produk rapi seperti simulasi di bawah ini.
Gunakan CSS manual Anda untuk meniru struktur ini (Card putih, background abu-abu, teks berwarna).

HABIS

*Ini hanya ilustrasi. Anda bebas berkreasi dengan warna dan layout di file style.css Anda.

Tugas 1: Siapkan Dekorasi (CSS Manual)

Kita tidak pakai CDN instan. Kita buat file CSS sendiri agar paham strukturnya.

Langkah A: Buka folder public di project Laravel Anda.
Langkah B: Buat folder css di dalamnya, lalu buat file style.css.
/* Copy CSS sederhana ini ke style.css */
body { background: #f9f9f9; font-family: sans-serif; padding: 20px; }
.product-card { background: white; border: 1px solid #ddd; padding: 15px; margin-bottom: 10px; border-radius: 5px; }
.price { color: green; font-weight: bold; }
.out-of-stock { color: red; font-weight: bold; }

Tugas 2: Sambungkan Kabel (Blade & Asset)

CSS sudah siap, tapi belum dipanggil oleh View.

1. Buka file resources/views/products/index.blade.php.

2. Di dalam tag <head>, tambahkan link CSS.

Wajib gunakan helper: {{ asset('css/style.css') }}

Cek di browser dengan "View Page Source", pastikan linknya mengarah ke URL yang benar.

Tugas 3: Logika Toko (Looping & If)

Tampilkan data produk dengan aturan bisnis berikut:

A. Looping: Gunakan @foreach untuk menampilkan semua produk dalam div class="product-card".
B. Format Uang: Harga jangan polos (15000). Gunakan number_format($product->price).
C. Logika Stok (@if):
  • Jika stok 0: Tampilkan teks "HABIS" (gunakan class .out-of-stock).
  • Jika stok ada: Tampilkan angka stoknya biasa.

Tugas 4: Info Kategori

Tampilkan nama kategori di bawah nama produk.

Syntax: {{ $product->category->name }}

*Jika muncul error "Attempt to read property 'name' on null", artinya ada produk yang category_id-nya tidak valid atau kosong.
