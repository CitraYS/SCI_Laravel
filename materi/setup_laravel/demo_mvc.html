Hands-on Lab

Implementasi Routes & MVC

Misi: Membuat Halaman "Profil Saya" yang dinamis menggunakan Controller.

Alur Data yang Akan Kita Buat

1. Route

Menerima URL /profil

2. Controller

Menyiapkan Data (Nama, Hobi)

3. View

Menampilkan HTML ke Layar

1

Buat Controller (Otak)

Kita butuh "Pelayan" untuk mengatur logika. Gunakan terminal untuk membuatnya secara otomatis.

TERMINAL (VS Code) bash
user@laptop:~/project$ php artisan make:controller ProfileController
Sukses jika muncul file baru di: app/Http/Controllers/ProfileController.php
2

Isi Logika Controller

Buka file controller yang baru dibuat. Kita akan membuat fungsi index untuk mengirim data nama kita ke View.

FILE: app/Http/Controllers/ProfileController.php
class ProfileController extends Controller { public function index() { // 1. Siapkan data (Biasanya dari Database) $nama_saya = "Budi Santoso"; $hobi = "Coding Laravel"; // 2. Kirim data ke View 'profile' // Kita pakai array asosiatif ['key' => $value] return view('profile', [ 'nama' => $nama_saya, 'hobi' => $hobi ]); } }
3

Buat View (Wajah)

Buat file baru di folder resources/views dengan nama profile.blade.php.

FILE: resources/views/profile.blade.php
<!DOCTYPE html> <html> <head> <title>Halaman Profil</title> </head> <body> <h1>Halo, Nama Saya {{ $nama }}</h1> <p>Hobi saya adalah: <b>{{ $hobi }}</b></p> <hr> <small>Dibuat dengan Laravel & MVC</small> </body> </html>
Perhatikan: Kita menggunakan kurung kurawal ganda {{ }} untuk mencetak variabel dari Controller. Ini adalah fitur Blade Template.
4

Daftarkan Route (Pintu)

Terakhir, kita harus memberitahu Laravel: "Kalau ada yang buka /profil, panggil Controller tadi".

FILE: routes/web.php
<?php use Illuminate\Support\Facades\Route; // 1. PENTING: Panggil Controller-nya di atas use App\Http\Controllers\ProfileController; Route::get('/', function () { return view('welcome'); }); // 2. Buat jalur baru ke Controller // Format: [NamaClass::class, 'NamaMethod'] Route::get('/profil', [ProfileController::class, 'index']);

Lihat Hasilnya!

Pastikan server menyala (php artisan serve), lalu buka browser.

http://localhost:8000/profil
