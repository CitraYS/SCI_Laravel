1. Routes (web.php)

Command: (Otomatis ada)
1
use Illuminate\Support\Facades\Route; Wajib
Library utama Routing. Tanpa baris ini, kode Route:: akan error.
2
use App\Http\Controllers\ProdukController; Wajib Import
Agar file ini kenal siapa itu "ProdukController". Wajib ditambah manual setiap bikin controller baru.
3
 
4
Route::get('/produk', [ProdukController::class, 'index']); Definisi
Struktur: Route::method('url', [Controller, 'fungsi']);
Gunakan get untuk tampil halaman, post untuk simpan data.
